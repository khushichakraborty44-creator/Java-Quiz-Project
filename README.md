# ☕ Java Mastery Quiz

An interactive and modern Java Quiz application built using **HTML, CSS, and JavaScript**. This project helps students, beginners, and interview aspirants strengthen their Java programming skills through topic-wise multiple-choice questions with instant feedback.

---

## 🚀 Features

- 🎯 80+ Java Interview & MCQ Questions
- 📚 Topic-wise Quiz Selection
- 🔀 Shuffle Questions
- 📊 Live Statistics
  - Total Questions
  - Answered Questions
  - Correct Answers
  - Accuracy Percentage
- ✅ Instant Answer Feedback
- 📈 Progress Bar
- 🏆 Final Score Screen
- 📱 Fully Responsive UI
- 🎨 Modern Glassmorphism Design
